# ToDo App ✅

Hi! I am **Mikołaj**, and this is my **ToDo App** built entirely by myself. 

I wrote this application using **React.js**, and I am very happy to finally show it to the world! This project represents a big step in my coding journey.

---

## 💡 About the Project

I am currently learning new things every single day. This app was a great way for me to practice React and prepare for the more advanced projects I will be building in the future.

### ✨ Features
* **Add Tasks:** Quickly create new things to do.
* **Complete Tasks:** Check off items as you finish them.
* **Delete Tasks:** Keep your list clean by removing old items.
* **Simple UI:** A clean and focused user interface.

### 🛠️ Built With
* **React.js** - The core framework
* **JavaScript (ES6+)** - Logic and interactivity
* **CSS3** - Custom styling and layout
* **HTML5** - Page structure
